# Code-Atlas

Code-Atlas helps developers understand complex codebases by visualizing
function calls, dependencies, and execution flow as diagrams.

## Features

- 📌 Roadmap view of code structure
- 🔗 Call graph visualization
- 🧭 Debug execution flow
- 🧠 Designed for students & junior developers

## How to Use

1. Open a project
2. Select a file or function
3. Open **Code-Atlas** from the sidebar
4. Explore the roadmap diagram

## Requirements

- VS Code 1.107.0 or later

## Extension Settings

No custom settings yet.

## Known Issues

- Large projects may take longer to analyze
